# Simple Bento Grid

A clean and responsive bento grid layout for displaying skills with icons.

## Features

- Responsive bento grid layout
- Various skill card sizes (small, medium, large)
- Hover animations
- Font Awesome icons
- Clean and modern design

## Technologies Used

- HTML5
- CSS3 (CSS Grid)
- Font Awesome 6

## Preview

The grid showcases skills like:
- Adobe Illustrator
- TypeScript
- NextJs
- React
- Tailwind
- SEO
- ExpressJs
- MongoDB
- Node.js
- And more...

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the skills in the HTML file
